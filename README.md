# Claude Code 一键安装程序

面向 Windows 的 Claude Code 商业化安装与配置工具（Tauri + React）。

## 功能

- 一键检测 / 安装 Node.js 与 Claude Code CLI
- 模型与 API 配置（官方 / 中转 / 本地推理）
- 终端代理、高级参数、备份恢复
- 实用工具：修复 PowerShell 策略、清理缓存、卸载

## 开发

```bash
npm install
npm run tauri dev
```

## 打包

```bash
npm run tauri build
```

## 版本

当前版本：`1.2.4`
